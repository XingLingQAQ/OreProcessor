# Ore Processor

Ore Processor is a plugin to customize mining experience.

## Features
- Fully-customizable
- Ore processing circuit
- Specialized storage for ores
- Offline-processing
- Upgrades
- Bulk operations
- Integration: ShopGUI+, EconomyShopGUI, AureliumSkills, Eco plugins
- Placeholders support
- JSON logging
